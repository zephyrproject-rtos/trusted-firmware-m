typedef uint16_t cyhal_gpio_mapping_cfg_t

Bitfield representing the configuration of a GPIO (hsiom selection and mode). Use the CY_GPIO_CFG_GET_MODE and CY_GPIO_CFG_GET_HSIOM to extract the individual field values.
