Hardware Abstraction Layer (HAL)
Typedefs
Port Specific Hardware Types

General Description

The following types are used by the HAL, but are defined by the implementation. As a result, the types can, and need to, be used in the application but the contents can not. It is not safe to rely on how the type is defined. In addition to changing between implementations of the HAL, they may also change between versions of a single implementation.
