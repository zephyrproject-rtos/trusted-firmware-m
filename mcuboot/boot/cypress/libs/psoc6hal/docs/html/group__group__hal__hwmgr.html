HWMGR (Hardware Manager)
HAL Drivers

General Description

High level interface for interacting with the Hardware Manager.

This provides two related functions:

  • Allows HAL drivers (or application firmware) to mark specific hardware blocks as consumed, so that other firmware will not accidentally try to use the block for a conflicting purpose.
  • For resources which are interchangeable, such as clock dividers, provides allocation and reservation of an available instance (if one exists).
